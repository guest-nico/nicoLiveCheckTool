## ニコ生放送チェックツール  
  
ニコ生のアラートツールです。  
RSS・ブラウザプッシュ通知・スマホアプリプッシュ通知から取得します。  
  
  
このツールで使用している[ニコ生のプッシュ通知の受信の手順](push.md)を記しました。
